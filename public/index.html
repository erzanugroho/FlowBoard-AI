Welcome to Flowboard AI

Your AI-powered kanban board that can read, write, and execute code in your projects.

1. Open a Project

Click Open Folder to select your project directory. Flowboard will store task data in a .flowboard/ folder inside it.

2. Add an AI Provider

Go to AI Providers in the sidebar. Add your OpenAI, Anthropic, or any OpenAI-compatible API key.

Use a model with tool/function calling support (GPT-4o, Claude 3.5+)

3. Create Tasks

Add cards to your board with a title and AI prompt. The prompt tells the agent exactly what to do.

Example: "Add input validation to the login form in src/auth.js"

4. Run the Agent

Click Run on a card or open the Agent panel. The AI will:

  • Read your project files
  • Plan the approach
  • Execute changes (with your approval)
  • Verify the result
